# LinearExpansionPack
Adds a collection of roles by LinearPoint to the game Demon Bluff by UmiArt

The new roles will show up in the Endless Mode for the game

**Important** In order to add art you will need to have the AreSwap mod by ITR13 and add the "role_LP" folders into the ArtSwap folder.

#### Roles
<ul>
  <li>***Pixie (Outcast)*** - Adds a fake minion to the Deck View and uses the abilities of the fake minion added.</li>
  <li>***Siren (Minion)*** - Appears as a Good Villager and both neighbors appear as an Evil Siren.</li>
</ul>
